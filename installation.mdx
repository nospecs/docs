# Installation

Follow these steps to install OVADARE and integrate it into your workflow.

## Prerequisites

Before installing OVADARE, ensure you have the following:

- Python version 3.7 or newer.
- pip, Python's package installer, installed on your system.
- Familiarity with multi-agent orchestration platforms like AutoGen or CrewAI.

## Step-by-Step Installation

### 1. Install OVADARE

Run the following command in your terminal to install OVADARE via pip:

```pip install ovadare```

This will install the latest version of OVADARE and its dependencies.

### 2. Verify Installation

To confirm OVADARE is installed, execute this command in your terminal:

```python -m ovadare --version```

You should see the installed version of OVADARE displayed.

### 3. Integrate OVADARE in Your Project

To start using OVADARE, import its modules into your Python scripts. For example:

```from ovadare import ConflictDetector, ResolutionEngine```

This allows you to begin integrating OVADARE’s conflict detection and resolution tools into your workflows.

## Troubleshooting Installation

If you encounter issues during installation:

- Ensure Python and pip are up to date.
- Check for permission errors and use ```--user``` with pip if needed:

```pip install --user ovadare```

- Refer to our [Troubleshooting Guide](troubleshooting) for detailed solutions.

Now that OVADARE is installed, dive into the [Quickstart](quickstart) to learn how to detect and resolve conflicts with minimal setup.
