## Installation Guide

Setting up OVADARE is simple and straightforward. Follow the steps below to install and configure OVADARE in your environment.

---

### 1. Prerequisites

Ensure you have the following installed:
- **Python** (version 3.7 or higher)
- **pip** (Python package manager)
- **git** (for cloning the repository)

To check if these tools are installed, run the following commands in your terminal:

```python --version```
```pip --version```
```git --version```

---

### 2. Install OVADARE from PyPI

The easiest way to install OVADARE is via `pip`. Run the command below:

```pip install ovadare```

This will download and install the latest version of OVADARE from the Python Package Index (PyPI).

---

### 3. Verify the Installation

Once installed, verify that OVADARE is working by checking its version:

```ovadare --version```

If the version is displayed, you're good to go!

---

### 4. Clone the OVADARE Repository (Optional)

For developers who want to explore or contribute to the OVADARE source code, you can clone the GitHub repository:

```git clone https://github.com/nospecs/ovadare.git```

Navigate to the repository directory:

```cd ovadare```

Then install the dependencies locally:

```pip install -e .```

---

### 5. Integrate OVADARE with Multi-Agent Platforms

OVADARE works alongside platforms like AutoGen and CrewAI. To integrate OVADARE, refer to the [Integration Guide](/how-to-guides/integrate-autogen).

---

### Troubleshooting

Having issues during installation? Check the following:
1. Ensure Python and pip are up-to-date:
   - ```python -m pip install --upgrade pip```
2. Verify your network connection if PyPI packages fail to download.
3. Consult the [community support page](/community) for help.
